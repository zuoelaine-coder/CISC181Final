# Nod Dash

Nod Dash is a motion-controlled browser game inspired by Geometry Dash.

---

## How It Works

* The webcam tracks the player's face and eye movement.
* A nod/head movement is detected using pose estimation.
* When a nod is detected:

  * the game starts
  * the player jumps over obstacles

The game uses:

* JavaScript
* HTML5 Canvas
* Kontra.js
* Webcam pose tracking

---

## Gameplay

* Nod to start the game
* Nod again to jump
* Avoid incoming obstacles
* The score increases the longer the user survives
* Press `R` to restart 

---

## Features

* Random obstacle spawning
* Scrolling background
* Sound effects and music
* Collision detection
* Score tracking
* Restart game functionality 
---

## Controls

| Action     | Control   |
| ---------- | --------- |
| Start Game | Nod       |
| Jump       | Nod       |
| Restart    | Press `R` |

## Technologies Used
